ویژن‌برد | Vision Board 

یک ویژن‌برد آفلاین و مینیمال ساخته‌شده با React و Vite با تمرکز بر
UI/UX لوکس، تعاملات نرم (iOS‑like) و طراحی.

این پروژه امکان مدیریت اهداف و ایده‌ها را به‌صورت گروه‌های چندتصویری
به‌همراه عنوان و یادداشت فراهم می‌کند، بدون نیاز به بک‌اند یا اینترنت.


امکانات

- پشتیبانی از چند تصویر برای هر ویژن
- افزودن عنوان و یادداشت
- ذخیره‌سازی کامل اطلاعات در LocalStorage
- رابط کاربری تیره، مینیمال و لوکس (بدون گرادیانت)
- micro‑interaction و افکت لمس نرم
- مودال ویرایش و حذف با تأیید کاربر
- سازگار با RTL و زبان فارسی
- ریسپانسیو (وب و موبایل)


تکنولوژی‌ها

- React
- Vite
- Tailwind CSS
- LocalStorage
